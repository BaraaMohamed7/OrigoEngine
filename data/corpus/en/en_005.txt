Software development involves designing, coding, testing, and maintaining software applications. Agile methodologies emphasize iterative development and customer feedback. DevOps combines development and operations to improve collaboration and deployment. Version control systems like Git help manage changes to code over time. Continuous integration and continuous deployment automate the software delivery process.

Software requirements define what the software should do. Software design creates the architecture and structure. Coding translates design into executable programs. Testing verifies the software meets requirements. Maintenance updates and improves the software after deployment. Waterfall follows sequential phases with minimal overlap. Scrum uses sprints to deliver working software incrementally. Kanban visualizes work to optimize flow. Extreme programming emphasizes technical excellence. Test-driven development writes tests before code. Pair programming has two developers work together. Code reviews improve code quality and share knowledge.
